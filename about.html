About This Project

This is a nutrition tracking website built using HTML, CSS, and JavaScript. It integrates with a backend API to help you track your food intake and calculate your daily nutritional needs.

Features:

  • Track food items with detailed nutritional information
  • Add, update, and delete food entries
  • Calculate your BMI and daily macro requirements
  • Get personalized nutrition recommendations based on your goals

Technologies:

  • Frontend: HTML, CSS, JavaScript (Fetch API)
  • Backend: ASP.NET Core Web API (MVC)
  • Database: SQL Server
